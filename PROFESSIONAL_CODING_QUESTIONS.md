## 📊 专业编程题一览

### 1️⃣ 购物车分析 - 实战编程：完整购物车分析函数
**题目**：补全购物车分析函数，完成数据清洗、指标计算和放弃率计算
**涉及知识点**：
- pandas数据清洗（dropna、fillna、to_datetime）
- nunique()统计唯一值
- 购物车放弃率计算
- 布尔字段过滤（purchase_completed == True）

### 2️⃣ 客户分群分析 - 实战编程：完整客户分群分析函数
**题目**：补全K-means聚类分析函数，完成数据标准化、聚类和结果分析
**涉及知识点**：
- StandardScaler数据标准化
- fit_transform()同时拟合并转换
- KMeans聚类器创建
- fit_predict()同时拟合并预测

### 3️⃣ 销售数据分析 - 实战编程：完整销售数据分析函数
**题目**：补全销售数据分析函数，完成时间序列处理、趋势分析和季节性分析
**涉及知识点**：
- pd.to_datetime()日期转换
- set_index()设置日期索引
- resample("ME")按月重采样
- pct_change()计算环比增长率
- .dt.month提取月份

### 4️⃣ 用户行为分析 - 实战编程：完整用户行为分析函数
**题目**：补全用户行为分析函数，完成会话识别、漏斗分析和路径分析
**涉及知识点**：
- 会话识别逻辑
- 转化漏斗计算
- groupby().agg()分组聚合
- nunique()统计用户数

### 5️⃣ A/B测试分析 - 实战编程：完整A/B测试分析函数
**题目**：补全A/B测试分析函数，完成数据分组、统计检验和结果解读
**涉及知识点**：
- ttest_ind()独立样本t检验
- Cohen's d效应量计算
- 显著性判断（p < 0.05）
- mean()计算转化率

### 6️⃣ 营销活动分析 - 实战编程：完整营销归因分析函数
**题目**：补全营销归因分析函数，实现首次触点归因模型
**涉及知识点**：
- sort_values()排序
- groupby("user_id").first()获取首次触点
- reset_index()重置索引
- value_counts(normalize=True)计算百分比

### 7️⃣ 库存管理分析 - 实战编程：完整库存分析函数
**题目**：补全库存分析函数，完成周转率计算、ABC分类和安全库存计算
**涉及知识点**：
- 库存周转率 = cogs / avg_inventory
- 周转天数 = 365 / 转动率
- cumsum()累计求和
- apply()应用自定义函数

### 8️⃣ 客户满意度分析 - 实战编程：完整满意度分析函数
**题目**：补全客户满意度分析函数，完成NPS计算、CSAT分析和情感分析
**涉及知识点**：
- NPS分类（推荐者/被动者/贬损者）
- apply()应用分类函数
- value_counts()统计分布
- CSAT满意率计算（rating >= 4）
- lambda函数进行情感分析

### 9️⃣ 竞品分析 - 实战编程：完整竞品分析函数
**题目**：补全竞品分析函数，完成市场份额计算、竞争力评分和市场定位
**涉及知识点**：
- 市场份额计算 = sales / total_sales * 100
- sort_values()排序
- 加权平均计算竞争力得分
- SWOT分析（识别优势）
- tolist()转换为列表

### 🔟 收入预测分析 - 实战编程：完整销售预测函数
**题目**：补全销售预测函数，完成时间序列处理、趋势分析和预测评估
**涉及知识点**：
- to_datetime()转换日期
- set_index()设置索引
- resample("ME")按月汇总
- rolling(window=3).mean()移动平均
- pct_change(12)计算同比增长率
- MAE和RMSE预测评估指标
